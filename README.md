### Getting Started

#### Configure

Edit constants at `pets_collection.tact`

```ts
const PetsCollectionName: String = "<Collection Name>";
const DefaultFallback: String = "<Fallback URI>";
```

#### Build and Test

```sh
npm install
npx blueprint build
npx blueprint test
```


#### Run Script
```sh
npx blueprint run
```
